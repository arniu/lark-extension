# ROADMAP

飞书云文档转 Markdown 小组件开发计划。按版本规划功能，每版聚焦一类能力。

---

## 版本规划概览

| 版本   | 主题     | 状态   |
| ------ | -------- | ------ |
| v0.1.0 | 核心转换 | 已完成 |
| v0.2.0 | 基础扩展 | 已完成 |
| v0.3.0 | 导出体验 | 规划中 |
| v0.4.0 | 高级功能 | 规划中 |
| v0.5.0 | 工程化   | 规划中 |

---

## v0.1.0 · 核心转换

文档 → Markdown 基础转换与 GFM 支持。

- [x] 文档导出为 Markdown（复制 / 下载）
- [x] 块类型：标题、正文、有序/无序列表、待办、代码块、引用、分割线、表格
- [x] GFM：表格、Task Lists、数学公式
- [x] 内联格式：粗体、斜体、行内代码、链接、删除线、下划线
- [x] 提及：@用户、文档引用、URL 预览、公式
- [x] 表格：cell 内容来源与表头/对齐正确输出
- [x] Task Lists：同级 TODO 合并为单个 list
- [x] 列表样式：统一 bullet 符号（避免 `-` / `*` 交替）

---

## v0.2.0 · 基础扩展

完善常用块与内联格式，提升导出完整度。

- [x] **IMAGE 块**：输出 `![alt](url)` 或 `![alt](url "title")`
- [ ] **代码块 language**：统一大小写（如 `JavaScript` → `javascript`）暂不实现
- [x] **文本颜色 / 背景色**：输出为 HTML span 或跳过

**发布建议**：上述项完成即可发 v0.2.0。

---

## v0.3.0 · 导出体验

扩展导出能力与选项。

- [ ] **导出格式**：GFM / CommonMark / 自定义
- [ ] **Frontmatter**：可选输出标题、日期等
- [ ] **图片处理**：内联 URL / 下载到本地
- [ ] **FILE 块**：输出 `[文件名](url)` 或占位说明
- [ ] **CALLOUT**：转引用块或带类型段落
- [ ] **DIAGRAM / MINDNOTE**：输出代码块或占位说明
- [ ] **BITABLE**：多维表格导出策略
- [ ] **QUOTE 单块**：与 QUOTE_CONTAINER 统一处理

**发布建议**：可按子集分批（如先 导出格式 + Frontmatter + 图片处理，再补其余块类型）。

---

## v0.4.0 · 高级功能

新方向与协作能力。

- [ ] **Markdown → 文档**：反向导入（md → BlockSnapshot）
- [ ] **批量导出**：多文档导出
- [ ] **导出过滤**：按块类型过滤
- [ ] **自定义映射**：块类型 → Markdown 映射配置

---

## v0.5.0 · 工程化

质量保障与交付流程。

- [ ] **单元测试**：docs-to-mdast 转换逻辑
- [ ] **快照测试**：GFM / Math 示例
- [ ] **CI**：构建与上传流水线
